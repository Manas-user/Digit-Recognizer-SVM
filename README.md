# 🧠 Digit Recognizer (MNIST) using SVM

## 📌 Overview
This project uses Support Vector Machine (SVM) to classify handwritten digits (0–9) from the MNIST dataset.

## 🚀 Techniques Used
- Support Vector Machine (Linear & RBF Kernel)
- One-vs-Rest multiclass classification
- PCA (Dimensionality Reduction)
- Model evaluation using accuracy

## 📊 Dataset
Kaggle Digit Recognizer Competition

## ⚙️ Results
- Linear SVM: ~91% accuracy
- RBF SVM: ~96% accuracy
- SVM + PCA: Faster with slight accuracy drop
